    실전 워크플로: 로드 → 전처리 → 분석 → 시각화 → GIS¶

    아래는 HOSSAM을 활용한 대표적인 작업 흐름 예시입니다.

    1) 데이터 로드¶

    from hossam import hs_util, data_loader

# 키 목록/설명 확인 (원격 메타데이터)
info = data_loader.load_info()
hs_util.pretty_table(info.head())

# 키로 데이터 로드
df = data_loader.load_data("AD_SALES")

    2) 전처리¶

    from hossam import hs_prep

# 카테고리 지정
from pandas import DataFrame
# df["category"] = df["category"].astype("category")  # 혹은 아래 헬퍼 사용
hs_prep.set_category(df, "category")

# 결측치 대체
df2 = hs_prep.replace_missing_value(df.select_dtypes(include="number"), strategy="mean")

# 스케일링(Standard)
scaled = hs_prep.standard_scaler(df, yname="y", save_path="std.pkl")

    3) 분석¶

    from hossam import hs_stats

# 다중공선성 제거(VIF)
filtered = hs_stats.vif_filter(df, yname="y", ignore=["id"], threshold=10.0)

# 추세선 계산
x = df["x"].to_numpy()
y = df["y"].to_numpy()
vx, vy = hs_stats.trend(x, y, degree=2)

    4) 시각화¶

    from hossam import hs_plot

hs_plot.scatterplot(df=df, xname="x", yname="y", hue="category", palette="Set1")
hs_plot.kdeplot(df=df, xname="x", hue="category", fill=True)

    5) GIS¶

    from hossam import hs_gis

# 지오코딩 (VWorld API 키 필요)
result = hs_gis.geocode(df, addr="address", key="YOUR_VWORLD_KEY")

# Shapefile 로드/저장
gdf = hs_gis.load_shape("path/to/file.shp")
hs_gis.save_shape(gdf, "out.gpkg")
