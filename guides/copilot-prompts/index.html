    Copilot Chat 프롬프트 예시 모음¶

    기본 동작: 코드 스니펫 반환. 실행이 필요하면 run:true(또는 mode:"run")를 명시하세요.

    코드만(기본)¶

    • "@hossam 이 DataFrame 결측치 분석하는 코드만 보여줘"
    • "@hossam 'Sales' 컬럼 정규성 검정 코드 예시 만들어줘"
    • "@hossam ./data.csv로 이상치 테이블(outlier_table) 코드만 생성해줘"
    • "@hossam KDE 플롯 그리는 코드만(데이터는 df, x='value', hue='group')"

    실행 결과까지 원할 때¶

    • "@hossam 실행해서 결과도 함께 보여줘" (또는 "run:true로 실행해줘")
    • "@hossam ./data.csv를 로드해서 category_summary 실행하고 결과를 출력해줘 (실행)"

    패턴별 템플릿¶

    • 분석: "@hossam [함수/작업] 코드만 생성해줘"
    • 실행: "@hossam [함수/작업] 실행해서 결과 출력해줘"
    • 파일 입력: "@hossam [파일경로]로 [함수][코드만/실행]"
    • 시각화: "@hossam [플롯함수] (인자들...) 코드만 / 실행"

    팁¶

    • 기본은 코드 반환입니다. 실제 실행이 필요하면 메시지에 "실행/결과"를 명시하거나 run:true를 붙여달라고 요청하세요.
    • 생성된 코드는 그대로 노트북/스크립트에 붙여넣어 사용할 수 있도록 구성됩니다.
